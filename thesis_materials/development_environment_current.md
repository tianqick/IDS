# 开发环境

本文系统开发与测试主要在 Windows 11 环境下完成，硬件配置能够满足 Web 系统开发、深度学习模型推理、数据预处理以及在线流量监测等需求。系统后端采用 Python 与 Flask 实现，数据库采用 MySQL，前端采用 Vue 3、Bootstrap 和 ECharts 完成页面交互与可视化展示，模型推理部分采用 PyTorch 实现。

## 环境说明

当前项目目录中存在 `.venv` 虚拟环境，主要依赖版本以该虚拟环境检测结果为准。项目后端通过 Flask 提供业务接口，使用 SQLAlchemy 与 MySQL 完成数据持久化；模型推理部分通过 PyTorch 加载训练完成的模型权重；在线流量监测部分通过 Scapy 完成抓包，并使用 Python cicflowmeter 进行流量特征提取。上述环境能够支撑系统的开发、调试、模型推理和功能展示。
